## Decisions Made

- **Crumble-intact = same color as normal**: The intact state of a crumble platform uses PLAT-1 (`#5a7a3a`) rather than a distinct color. Players cannot identify crumble platforms visually until they land on one. This matches the game mechanic: surprise is intentional.
- **imageSmoothingEnabled set once at init**: Not inside save/restore blocks, because browser behavior for this property across save/restore is inconsistent. Set once, rely on it globally.
- **GPL palette format**: Chosen because Pixelorama imports it natively via the Palettes menu. Hex values in STYLE_GUIDE.md and RGB values in the .gpl file are in sync.
